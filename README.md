# Cheepuri Jyothi Swaroopan - Portfolio

A beautiful, 3D interactive portfolio website showcasing Business Analytics, Data Science, and Machine Learning projects. The design features a modern glassmorphism aesthetic, 3D hover effects, and responsive design tailored for a data-driven professional.

## 🚀 Live Demo
This portfolio is ready to be hosted on GitHub Pages or any standard static file host.

## ✨ Features
- **Modern Glassmorphism Design:** Sleek translucent panels with animated gradient blob backgrounds.
- **3D Interactive Project Cards:** Immersive hover interactions engineered using Atropos.js.
- **Responsive Layout:** Perfectly scales across mobile devices, tablets, and desktops.
- **Projects Displayed:**
  - **Revenue Optimizer:** Dynamic pricing machine learning model.
  - **Supply Chain Commander:** Predictive analytics dashboard.

## 🛠️ Built With
- **HTML5 & Vanilla CSS3**
- **Atropos.js** for 3D parallax hover effects
- **Lucide Icons** for scalable vector graphics
- **Google Fonts** (Outfit)

## 💻 Local Setup
1. Clone this repository: 
   ```bash
   git clone https://github.com/swaroopan-crypto/Portfolio-Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio-Website
   ```
3. Open `index.html` directly in any modern web browser or run a local Live Server.
